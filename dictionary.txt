exponential - Exponential functions are functions of that have a fixed base b that could be any positive real number. Exponential functions are often associated with exponential growth, which means that that there is a consistent fixed period over which the function will double (or triple, or quadruple, etc.) and the change is always a fixed proportion.
function - The function is a general relation between a set of input values (domain) and corresponding set of output values (co-domain). It relates each element of a set (domain) with exactly one element of another set (co-domain). Every element in domain must have a corresponding value in co-domain. A function is single valued. It can't give two or more output values for the same input.
